# 📖 Acerca del proyecto

Este proyecto es un **fork no oficial** del excelente trabajo realizado por **ccloli**.

Con el paso del tiempo, el sitio web y los navegadores modernos fueron cambiando, provocando que algunas funciones dejaran de funcionar correctamente.

Este repositorio nace con un único objetivo:

> **Mantener la extensión funcional y compatible con los navegadores actuales.**

No pretende sustituir el proyecto original, sino darle continuidad mediante correcciones, mantenimiento y futuras mejoras.

# 📦 Instalación

## Chrome / Opera / Edge

1. Descarga o clona este repositorio.

2. Abre:

chrome://extensions

3. Activa **Modo desarrollador**.

4. Selecciona:

**Cargar extensión descomprimida**

5. Elige la carpeta del proyecto.

---

## Firefox

Abre:

about:debugging

Selecciona:

**Este Firefox**

Haz clic en:

**Cargar complemento temporal**

Selecciona el archivo:

manifest.json
